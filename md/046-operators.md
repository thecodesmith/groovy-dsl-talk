## Operator Overloading

```
| Code   | Method            |
| a + b  | a.plus(b)         |
| a - b  | a.minus(b)        |
| a * b  | a.multiply(b)     |
| a / b  | a.divide(b)       |
| a % b  | a.modulo(b)       |
| a ** b | a.power(b)        |
| a | b  | a.or(b)           |
| a & b  | a.and(b)          |
| a ^ b  | a.xor(b)          |
| a[b]   | a.getAt(b)        |
| a << b | a.leftShift(b)    |
| a >> b | a.rightShift(b)   |
| +a     | a.positive()      |
| -a     | a.negative()      |
| ~a     | a.bitwiseNegate() |
```
